Software User Guide

1. Daily Income/Expense

Rozana ki sale ke liye "Income Sale" select karein. Shop ke kharchon ke liye "Expense" select karein. Cash Balance khud ba khud update hoga.

2. Borrowing (Loan Taken)

Agar kisi se karz liya hai to "Borrowing" select karein. Dashboard par "Loan Bal" barh jayega. Jab wapas karein to Expense module use karein.

3. Adjustment/Loss

Agar cash kam ho jaye ya nuqsan ho, to "Loss/Adjustment" select karein. Ye amount total cash se minus ho jayegi.

4. Search & Print

Kisi khas bande ka ledger dekhne ke liye Search box mein uska naam likhein. Phir "Print" dabayein to sirf uska statement print hoga.
