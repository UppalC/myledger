MyLedger User Guide

1. Account banana

Step 1: Website open karein.

Step 2: New user? Create account par click karein.

Step 3: Apna email likhein.

Step 4: Password likhein.

Step 5: Terms parh kar I Read & Agree par click karein.

2. Login aur password reset

Login: Email + password likhein aur Login par click karein.

Forgot password: Email likhein, phir Forgot password? par click karein. Reset email inbox/spam mein aayegi.

3. Easy mode mein entry

Step 1: Date select karein.

Step 2: Party ya naam likhein.

Step 3: Category select karein.

Step 4: Amount likhein.

Step 5: Save Entry par click karein.

4. Search aur reports

Search: Kisi bhi lafz se record dhoondne ke liye.

Party Report: Kisi aik banday ka poora khata nikalne ke liye exact naam likhein.

Report Type: Full, Income, Expense, Udhaar, Clinic, Loan.

Quick Reports: Aaj, Is Haftay, Is Mahine, Sab.

Rule: Jo report screen par hai, wahi print, Excel aur WhatsApp share mein jayegi.

Loan samajhne ka asaan rule: Agar aap ne paisa liya ho to Borrowing / Loan Taken use karein. Agar aap ne liya hua paisa wapas kar diya ho to Loan Repaid / Wapas Kiya use karein.

Udhaar samajhne ka rule: Agar aap ne kisi ko paisa diya ho to Udhaar Diya. Agar us ne wapas kar diya ho to Udhaar Receive.

5. Settings guide

Mode: Easy = simple, Advanced = extra details, Expert = full control.

Theme color: Apni pasand ka color select karein.

Font size: Chhota ya bara text.

Dark mode: Raat ke liye.

Categories: Har category ka naam, type aur visibility change kar sakte hain.

Category type: income / expense / receivable_in / receivable_out / loan_in / loan_out / clinic_bill / clinic_recovery / other

Asar: Category type badalne se dashboard aur reports ka behavior bhi badalta hai.

6. Backup aur restore

Backup: Backup button par click karein, password dein, file save karein.

Restore: Restore par click karein, file select karein, password dein.

Factory reset: Reset se pehle software aap ko backup lene ki warning dega.
