MyLedger User Guide

1. Login ya signup ka tareeqa

Email aur password likhein. Agar naya account hai to “Create account” par click karein, terms read karke account bana lein.

2. Nayi entry kaise karein

  • Date select karein
  • Payment mode select karein
  • Party ya person ka naam likhein
  • Category choose karein
  • Amount likhein
  • Save Transaction par click karein

3. Ghar ke istemal ke liye

Expense category ko ghar kharcha, ration, bijli, school fee ke liye use karein. Party field mein item ka naam ya family member ka naam likh sakte hain.

4. Dukan ya worker ke liye

Income Sale se aamdani record karein. Expense se kharcha. Udhaar diya aur udhaar receive alag categories mein record karein.

5. Visit / Clinic module

Visit/Clinic Pro choose karne par extra box khulta hai. Us mein visit type, reason aur medicine cost alag save hoti hai.

6. Edit aur delete

Table mein edit icon par click karke entry badal sakte hain. Delete icon se entry hategi.

7. Search aur filter

Search se party, category, date aur notes par filter lagta hai. Date range se specific period dekh sakte hain.

8. Backup aur restore

Backup par click karke password dein. File .ledger format mein download hogi. Restore ke liye file select karein aur wahi password dein.

9. Settings

Settings se date format, software title, categories ke names, colors aur visit options badal sakte hain.

10. Print

Print button se current table print hogi. Search/filter ke baad print karenge to wahi report niklegi.
